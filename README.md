# kindle-tools

This is a collection of tools for jailbroken kindles.

As of now, the list includes:

- Wallpapers tool for managing screensaver presets. A web inteface
  lets you create presets, upload wallpapers, and chose the applied preset.
- JsRepl tool for a remote js repl on the kindle, useful for testing
  interactively javascript behavior on the Mesquite webview.
